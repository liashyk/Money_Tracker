# Money Tracker - Система обліку фінансів

Це веб-додаток для ведення особистого бюджету. Він дозволяє створювати гаманці в різних валютах, додавати витрати та доходи, а також переглядати статистику на графіках. Проект використовує актуальні курси валют НБУ.

Технології: PHP 8, MySQL, JavaScript, Docker.

----------------------------------------------------------------
ЯК ЗАПУСТИТИ ЧЕРЕЗ DOCKER
----------------------------------------------------------------

Вам не потрібно встановлювати веб-сервер або базу даних вручну. Потрібен лише Docker Desktop.

1. Завантажте папку з проектом.
2. Відкрийте термінал (або командний рядок) всередині цієї папки.
3. Виконайте команду:

   docker-compose up -d --build

4. Зачекайте хвилину, поки система завантажиться.
5. Відкрийте браузер за адресою:

   http://localhost:8888

(Якщо ви не змінювали налаштування порту).

----------------------------------------------------------------
ЯК ЗАРЕЄСТРУВАТИСЯ ТА УВІЙТИ
----------------------------------------------------------------

У системі працює спрощена авто-реєстрація. Окремої кнопки "Реєстрація" немає.

1. Придумайте собі Логін та Пароль.
2. Введіть їх у форму входу на головній сторінці.
3. Якщо такого користувача ще немає в базі — система автоматично створить його, і ви одразу увійдете.
4. Якщо такий користувач вже існує — система перевірить пароль.

Приклад для першого входу:
Логін: admin
Пароль: 12345

----------------------------------------------------------------
ВИРІШЕННЯ ПРОБЛЕМ (ПОМИЛКА 404)
----------------------------------------------------------------

Якщо після запуску ви бачите помилку 404 Not Found або сайт не відкривається, перевірте наступне:

1. Переконайтеся, що ви запускали команду docker-compose up, знаходячись саме всередині папки з проектом, де лежать файли index.php та docker-compose.yml.

2. ПРОБЛЕМА З ПОРТОМ:
Часто буває, що порт 8888 або 8080 вже зайнятий іншою програмою на вашому комп'ютері.
Щоб це виправити:
 - Відкрийте файл docker-compose.yml у будь-якому текстовому редакторі.
 - Знайдіть рядок, де написано "ports:".
 - Змініть значення, наприклад, з "8888:80" на "9090:80".
 - Збережіть файл.
 - Перезапустіть проект командами:
   docker-compose down
   docker-compose up -d
 - Відкрийте сайт за новою адресою: http://localhost:9090

----------------------------------------------------------------
КОРИСНІ КОМАНДИ
----------------------------------------------------------------

Зупинити проект:
docker-compose down

Зупинити проект та видалити базу даних (повне очищення):
docker-compose down -v
